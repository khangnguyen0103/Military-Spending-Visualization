# Military Spending Visualization

This project explores global military spending as a share of GDP across countries. It uses interactive visualization to make international defense spending patterns easier to compare, interpret, and communicate.

The main visualization includes a choropleth map for country-level comparison and a trend-based view for understanding how military spending changes over time. Instead of presenting the data as a static table, the project turns it into a visual story that helps readers quickly identify high-spending regions, compare countries, and notice broader global patterns.

## Project Focus

- Compare military spending across countries
- Visualize spending as a percentage of GDP
- Identify regional and global spending patterns
- Communicate complex international data through interactive charts

## Tools Used

- R
- HTML
- Plotly
- Data visualization
- Choropleth mapping
